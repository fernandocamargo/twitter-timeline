# Twitter Timeline

## Installation

### Using Docker (Recommended)

```bash
docker-compose up
```

The application will be available at:
- Frontend: http://localhost:8080
- Backend API: http://localhost:1337

### Using Local Environment

```
npm install
```

## Development

### With Docker

```bash
docker-compose up
```

### Without Docker

```
npm start
```

## Build

```
npm run build
```
